# Maintainers

This file lists the maintainers of the Hearth project. Maintainers review and merge contributions,
triage issues, and steer the roadmap.

## Becoming a maintainer

Hearth is early and actively looking for maintainers. Consistent, high-quality contributions
(code, reviews, docs, issue triage) are the path in — open a discussion or reach out if you'd like to
get more involved. See [`CONTRIBUTING.md`](CONTRIBUTING.md).

## Security

To report a vulnerability, **do not** use this list for an initial private report — follow
[`SECURITY.md`](SECURITY.md).
